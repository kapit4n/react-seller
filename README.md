# react-seller
## Pre-requisites
* node 6
* npm
* loopback

## Install client
* clone repo
* cd projectPath/client
* npm install
* npm start

## Install Sever
* clone repo
* projectPath/server
* npm install
* node .

## Install Sever
* GET /products
* GET /stock
* GET /sells
* GET /customers

* POST /products
* POST /stock
* POST /sells
* POST /customers

* PUT /products
* PUT /stock
* PUT /sells
* PUT /customers
